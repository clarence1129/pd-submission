My Projects

A collection of creative and technical work

VR Escape Room Unity

An immersive, multi-puzzle experience designed and developed in Unity. Players navigate a virtual environment where they interact with objects, solve logical puzzles, and uncover hidden secrets to progress through the game. Features include teleportation mechanics, realistic object interactions, and carefully crafted puzzle logic.

This project demonstrates my ability to design interactive gameplay, implement VR mechanics, and create a cohesive user experience that is both fun and engaging.

E-Commerce Website Prototype

A minimalistic e-commerce prototype designed using Figma with a clean, modern, and intuitive shopping experience. Includes fully structured layouts for product pages, shopping cart interactions, and navigation flows that prioritize usability and visual clarity.

This project highlights my ability to translate design concepts into interactive prototypes, plan seamless user flows, and create visually appealing interfaces ready for development.

Notes Website

A full-stack note management platform with user authentication, note creation/editing/deletion, and support for file uploads. Users can register accounts, securely log in, and organize their notes with photos and documents.

Built with React frontend and Java backend, this project demonstrates modern web development practices including authentication flows, API integration, and responsive design across all devices.

Learn@RP

A comprehensive academic management application designed to help students track learning progress, manage assignments, and stay informed about exam dates. Features intuitive dashboards and progress analytics.

This project showcases my UI/UX design expertise with complete design systems, interactive prototypes, and user-centered interface design. Built with a focus on usability and student needs.
