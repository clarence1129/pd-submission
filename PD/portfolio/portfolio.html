My Projects

VR Escape Room Unity

The VR Escape Room is an immersive, multi-puzzle experience designed and developed in Unity. Players navigate a virtual environment where they interact with objects, solve logical puzzles, and uncover hidden secrets to progress through the game. The project incorporates teleportation mechanics, realistic object interactions, and carefully crafted puzzle logic to create an engaging and interactive experience.

Every puzzle is designed to challenge problem-solving skills while maintaining a smooth and intuitive flow, ensuring that players feel both challenged and rewarded. The environment is fully immersive, with attention to visual aesthetics, sound design, and spatial interactions, making it a compelling VR experience that combines creativity, design thinking, and technical development.

This project demonstrates my ability to design interactive gameplay, implement VR mechanics, and create a cohesive user experience that is both fun and engaging.

Figma Website

The Shopping Website (Figma) is a minimalistic e-commerce prototype designed to deliver a clean, modern, and intuitive shopping experience. Using Figma, I created a fully structured layout including product pages, shopping cart interactions, and navigation flows that prioritize usability and visual clarity.

The project focuses on user-centered design, ensuring that every element — from product images and descriptions to buttons and menus — guides the user naturally through the shopping journey. Color schemes, typography, and spacing were carefully chosen to create a modern and cohesive aesthetic.

This project highlights my ability to translate design concepts into interactive prototypes, plan seamless user flows, and create visually appealing interfaces that are ready for development. It demonstrates my skills in UI/UX design, prototyping, and digital storytelling, offering a professional and engaging shopping experience.
