Notes Website

A full-stack note management platform with authentication and file uploads

Project Overview

A comprehensive note management web application that enables users to create, store, edit, and delete notes with rich media support. Users can register accounts, log in securely, and manage their personal note collection with the ability to upload photos and documents.

Built with a modern tech stack combining React for the frontend and Java for the backend, this application provides a seamless user experience with robust authentication and file handling capabilities.

Key Features

  • User Registration & Authentication - Secure account creation and login system
  • Note Management - Create, read, update, and delete notes with ease
  • File Uploads - Support for photos and document uploads to enhance note content
  • Rich Editor - User-friendly interface for composing and formatting notes
  • Responsive Design - Fully responsive across desktop, tablet, and mobile devices
  • Real-time Updates - Instant feedback on note creation and modifications

Technologies & Tools

React Java REST API Authentication File Upload Bootstrap Responsive Design

My Role

I was responsible for developing the entire frontend interface of this application. I created an intuitive and visually appealing user interface using React, ensuring smooth interactions for note management, file uploads, and user authentication flows. The frontend design prioritizes user experience with clear navigation, responsive layouts, and efficient state management.
